# Star Wars Character Explorer

A responsive web application that displays Star Wars characters using the SWAPI (Star Wars API). Built with React, Tailwind CSS, and modern web development practices.

## Features

## Core Features
-  Fetch and display Star Wars characters from SWAPI
-  Pagination support for browsing all characters
-  Loading and error states
-  Character cards with random images and species-based colors
-  Modal view with detailed character information
-  Homeworld details (name, terrain, climate, population)
-  Fully responsive design (mobile, tablet, desktop)

### Bonus Features
-  Search by character name (partial match)
-  Filter by homeworld, film, and species
-  Combined search and filter functionality
-  Mock authentication with login/logout
-  Silent token refresh mechanism
-  Integration tests for modal functionality

## Technologies Used

- **React** (v18+ with hooks and functional components)
- **Tailwind CSS** for styling
- **HTML5 & CSS3**
- **JavaScript (ES6+)**
- **SWAPI** - Star Wars API
- **Picsum Photos** for random character images

# How to Run 
- Clone the Repository 
- Install dependencies
- Start the development server
